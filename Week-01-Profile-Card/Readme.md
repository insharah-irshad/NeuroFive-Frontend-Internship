# Week 01 - Personal Profile Card

## Project Overview

This project is my Week 1 task for the NeuroFive Solutions Frontend Web Development Internship.
I created a personal profile card using HTML and CSS. The purpose of this project was to practice HTML structure, CSS styling, typography, colors, and responsive design without using JavaScript or any CSS framework.


## Features
1. Personal profile photo
2. Name and short introduction
3. Three social links with icons
4. Clean card-based UI design
5. Responsive layout for different screen sizes
6. Custom styling using pure CSS


## Technologies Used
1. HTML
2. CSS
3. Google Fonts
4. Font Icons


## Design & Styling

The profile card was designed with:
1. Semantic HTML elements like header, section, and footer
2. CSS box model concepts using margin and padding
3. Border-radius for smooth edges
4. Box-shadow for depth and modern appearance
5. Custom typography using multiple font styles
6. Responsive design using CSS media queries

## Responsiveness

The card layout was tested for different screen sizes:
1. Small screens (350px)
2. Larger screens (500px)

## CSS Challenges

While creating this profile card, I found Flexbox alignment and spacing properties slightly challenging. Understanding the CSS box model, especially margin and padding, required some practice and adjustments. I solved these challenges by testing different values and using browser developer tools to improve the layout. This project helped me better understand responsive design, typography, and clean CSS styling.

## What I Learned

Through this project, I improved my understanding of:
1. Writing clean semantic HTML
2. Structuring a frontend project
3. Using CSS properties effectively
4. Creating responsive layouts
5. Combining fonts and colors for better UI design
